# AFK System (離席演出アプリ)

離席中（AFK: Away From Keyboard）であることを、スタイリッシュに画面に表示するWindows用アプリケーションです。
スクリーンセーバーのように動作し、動く文字と稲妻エフェクトで画面を彩ります。

## ⚡ 特徴

* **カスタムテキスト**: 「お風呂」「ご飯」「仮眠」など、好きな文字を表示して離席できます。
* **履歴機能**: 過去に入力した文字を記憶し、サッと選べます。
* **動的なエフェクト**: 文字が画面内を跳ね回り、ランダムで稲妻が走ります。
* **自動アップデート**: 起動時に最新版があるかチェックし、自動で通知します。

## 📥 ダウンロード

右側の「Releases」から最新のインストーラー（`AFK Setup.exe`）をダウンロードしてください。

[👉 最新版のダウンロードはこちら](https://github.com/Ryu-62/AFK/releases)

## 🎮 使い方

1.  インストールした `AFK` を起動します。
2.  入力画面が表示されるので、離席理由（例：「お風呂」「コンビニ」）を入力します。
    * 何も入力せずにOKを押すと「お風呂.exe」になります。
3.  `OK` ボタンまたは `Enter` キーで開始！
4.  **終了方法**: `Esc` キーを押すか、画面をクリックすると終了します。

## 🛠 動作環境

* Windows 10 / 11
